# icontext-visualize - Gemini CLI adapter

Point Gemini at `PROCEDURE.md` in this folder. It should confirm `CONTEXT.md` exists, run the
bundled `scripts/export-context-graph.js` exporter, create static `context-graph.html` and
`context-graph.js` files, verify them, report output paths plus graph counts, and stop without
editing source context or committing.
